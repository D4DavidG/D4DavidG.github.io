FitnessFunctions

Role-based class enrollment system · PHP + MySQL · Admin & Member dashboards

GitHub ↗  ·  Download Project ZIP

Overview

FitnessFunctions is a role-based web app that manages gym instructors, class schedules, and member enrollments. Admins can create/edit/delete instructors and classes, while members can browse upcoming sessions, enroll, and drop classes. Enrollment counts are tracked per class to support capacity limits.

Key Features

  • Authentication + Roles: users log in and are routed to Admin or Member dashboards.
  • Admin Dashboard: manage instructors, manage classes, and view enrollments.
  • CRUD Operations: add/edit/delete instructors and classes.
  • Enrollment System: members enroll from “All Classes” and drop from “My Classes”.
  • Capacity Tracking: “Enrolled” count updates and classes can show Full.

Technologies

  • Frontend: HTML, CSS (custom neon dashboard UI)
  • Backend: PHP
  • Database: MySQL
  • Environment: Apache (localhost development)
